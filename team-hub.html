خطوات إعداد النظام الكامل للفريق
01

إنشاء Google Drive Folder هيكل

اعمل هيكل موحد على Google Drive — ده مكان رفع الملفات الثقيلة

📁 Team Hub — Drive
├── 📁 01_Projects
│ ├── 📁 حملة_رمضان_2025
│ │ ├── 📁 Raw_Files
│ │ ├── 📁 Drafts
│ │ └── 📁 Final ✅
│ └── 📁 مشروع_X
├── 📁 02_Archive ← أرشيف تلقائي
├── 📁 03_Assets ← لوجوهات وموسيقى
└── 📁 04_Reports ← تقارير سوشيال
  • شارك الـ folder مع الفريق بصلاحيات محددة
  • إنتاج → Editor في Projects فقط
  • سوشيال → Viewer على Final folders
  • مسؤولين → Viewer على Archive بعد موافقتك
02

إعداد Notion Task Database

أنشئ Database بالـ properties دي — ده قلب النظام

Task Name → Title
Status → Backlog | In Progress | Review | Production | Done
Type → Video | Graphic | Content | Social | AI
Assignee → Person
Deadline → Date
Priority → 🔴 Urgent | 🟡 High | 🟢 Normal
Drive Link → URL ← رابط الـ folder على Drive
Platform → Instagram | TikTok | YouTube
Archive Date → Date ← يتملي تلقائي لما Status = Done
03

ربط Workflow الإنتاج والسوشيال

الخطوات اللي بيتبعها الفريقين مع بعض

  • الإنتاج بيرفع الفيديو في Drive على Drafts folder
  • بيحدث الـ Status في Notion لـ "In Review" + بيضيف Drive Link
  • أنت بتراجع وتعمل Approve → Status يتحول لـ "Production"
  • الإنتاج بينقل الملف لـ Final folder
  • السوشيال بيشوف الـ task "Production" ويحمل من Final
  • بعد النشر: Status → Done وبيتنقل تلقائي للـ Archive
04

نظام الأرشيف التلقائي

خلي كل حاجة تتأرشف أوتوماتيك بدون أي جهد

  • في Notion: اعمل Filtered View اسمه "Archive" يظهر Tasks اللي Status = Done
  • في Drive: استخدم Google Drive Automations أو Zapier عشان ينقل الـ Final files للـ Archive folder تلقائياً
  • كل ملف في الأرشيف يحتفظ بالـ task details والـ Drive link
  • المسؤولين والعملاء يقدروا يوصلوا للأرشيف بعد موافقتك فقط
05

إدارة وصول المسؤولين والعملاء

إزاي تدي أي حد وصول مؤقت أو دائم للبيانات

  • Notion: شارك صفحة معينة كـ "View Only" — مش الـ workspace كله
  • Drive: شارك الـ folder المخصوص بـ Viewer permission فقط
  • اعمل Guest Account في Notion للعملاء — ما يشوفوش إلا اللي تختاره
  • كل 30 يوم: راجع الـ shared accesses وشيل اللي انتهى
  • في Notion يمكنك عمل Password-protected page للعملاء الكبار
